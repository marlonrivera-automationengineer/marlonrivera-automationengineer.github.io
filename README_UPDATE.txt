HOW TO UPDATE YOUR LIVE GITHUB PAGES WEBSITE

1. Extract this ZIP file.
2. Open your GitHub repository:
   marlonrivera-automationengineer.github.io
3. Delete or replace the existing:
   index.html
   styles.css
   script.js
   assets folder
4. Upload the new index.html, styles.css, script.js and complete assets folder.
5. Commit the changes.
6. Wait approximately 1–5 minutes, then refresh the website using Ctrl+F5.
