Vacuum Monitoring & Predictive Maintenance System

Centralized predictive-maintenance platform with real-time dashboards, trends and alarms.

Vacuum sensorsPythonFastAPIHTMLJavaScriptSQLPredictive maintenance
OverviewResponsibilitiesChallengeSolutionImpactLessons

Overview

Engineering context

Designed a centralized platform providing real-time vacuum pressure, machine health, historical trends and alarm visibility.

Responsibilities

My contribution

System architecture, data collection, Python software, database, dashboards, alarm management, analysis and deployment.

Challenge

What needed to be solved

Manual machine-side observation made gradual degradation and historical fault investigation difficult.

Solution

Engineering approach

Built a web platform with configurable machines, real-time charts, alarm logging, records and centralized maintenance visibility.

Impact

Project outcomes

  • Real-time monitoring
  • Faster fault investigation
  • Improved preventive maintenance
  • Centralized records

Lessons Learned

Engineering reflection

This project reinforced the importance of validating process requirements before finalizing the machine concept, involving production and maintenance teams early, and confirming performance through structured testing before handover.
