HOW TO PUBLISH
1. Extract this ZIP.
2. Open:
   D:\GitHub\marlonrivera-automationengineer.github.io
3. Keep the hidden .git folder.
4. Copy every extracted item into the repository folder.
5. Choose Replace the files in the destination.
6. Test index.html locally.
7. Open GitHub Desktop.
8. Summary:
   Publish final portfolio website Version 3.0
9. Click Commit to main.
10. Click Push origin.
11. Wait 1–5 minutes.
12. Open the live website and press Ctrl+F5.
