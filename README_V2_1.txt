HOW TO PUBLISH

2. Copy every item into:

3. Keep the hidden .git folder.
4. Replace existing files when Windows asks.
5. Test index.html locally.
6. Open GitHub Desktop.
7. Summary: Upgrade portfolio website to Version 2.1
8. Commit to main.
9. Push origin.
10. Wait 1–5 minutes and press Ctrl+F5 on the live site.
