=============================

FILES

index.html   Main webpage

assets/     Resume and future project images

HOW TO PREVIEW
--------------
Double-click index.html. It will open in your web browser.

HOW TO PUBLISH WITH GITHUB PAGES
--------------------------------
1. Create a free GitHub account.
2. Create a new PUBLIC repository named:
   yourusername.github.io
3. Upload all files and the assets folder.
4. Open repository Settings > Pages.
5. Under Build and deployment:
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
6. Save and wait a few minutes.
7. Visit:
   https://yourusername.github.io

IMPORTANT
---------
Only publish non-confidential photos, drawings, customer names, machine details,
PLC code, production data, and cost information that you are legally allowed to share.
